Why are developers obsessed with Scott Adams on GitHub?

Decades after the original Scott Adams text adventures defined an era of interactive fiction, developers on GitHub are reverse‑engineering engines, documenting file formats, and building new tools to keep these games alive. This website explores that obsession: the projects, the people, and the patterns behind the revival.
